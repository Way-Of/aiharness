# Example Skill for Codex CLI

This is the prompt.md companion file for the Codex two-file skill format.

## Requirements
- Directory name: `example_skill` (snake_case)
- `name` in skill.yaml matches directory name
- `allowed-tools` values are lowercase_snake (e.g., `read_file`, `write_file`)
- Config directory: `~/.codex/skills/`
